## 🏃‍♂️ Quick Start

### 1. Navigate to Web Directory
```bash
cd web
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Application
```bash
npm start
```

### 4. Open in Browser
The app will automatically open at `http://localhost:3000`
